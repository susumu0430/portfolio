# SusumuMind — Portfolio

susumu のポートフォリオサイト。
**AI Imagery × Structured Thinking** — 思考法と生成AIを掛け合わせ、どちらにもない価値を生み出す。

## サイト構成

| セクション | 内容 |
|-----------|------|
| Hero | Structured Mind × Infinite Vision |
| About | 3つの強み（構造化思考 / AI画像制作 / 組み合わせる力） |
| Gallery | AI画像作品（リアル系キャラクター） |
| Tools | 使用ツール（Higgsfield AI ほか） |
| Links | X / Pixiv / note / YouTube |
| Contact | メール・X DM |

## 技術スタック

- [Next.js](https://nextjs.org) (App Router) + TypeScript
- [Tailwind CSS v4](https://tailwindcss.com)
- [Framer Motion](https://www.framer.com/motion/) — スクロール連動アニメーション

## 開発

```bash
npm install
npm run dev    # http://localhost:3000
npm run build  # 本番ビルド
```
